# AgriAssist

AgriAssist is an AI-powered smart farming companion designed to help farmers optimize their crop yields, monitor farm health, and access real-time market insights.

## Features

*   **Crop Disease Scanner**: AI-based detection of crop diseases from photos.
*   **Smart Irrigation Advisor**: Recommendations for optimal water usage.
*   **Market Insights**: Real-time price tracking and demand analysis.
*   **Weather Intelligence**: Localized weather forecasts and alerts.
*   **Farmer Registration**: Digital identity for farmers.

## Technology Stack

*   React + Vite
*   TypeScript
*   Tailwind CSS
*   Shadcn UI
*   Supabase (Backend/Auth)
*   Node.js/Express (API)

## Getting Started

1.  Navigate to the `frontend` directory:
    ```bash
    cd frontend
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the development server:
    ```bash
    npm run dev
    ```

## Deployment

This project is configured for deployment on Vercel (Frontend) and Render/Heroku (Backend).
